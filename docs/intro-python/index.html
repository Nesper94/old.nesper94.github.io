Para ver esta introducción a Python descarga este archivo y descomprímelo. Adentro encontrarás el notebook intro-python.ipynb con ejercicios prácticos diseñados principalmente para biólogos. Para ver este archivo necesitarás instalar primero Jupyter siguiendo las siguientes instrucciones:

Instalando Jupyter

Linux

Para instalar Jupyter en Linux, ejecuta el siguiente comando:

sudo apt install jupyter-notebook

Para ejecutar Jupyter:

jupyter-notebook

Windows

En Windows se recomienda instalar Anaconda ya que instala automáticamente Jupyter y otras dependencias y paquetes de software usados en ciencia de datos.
